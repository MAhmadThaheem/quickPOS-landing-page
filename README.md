# QuickPOS Landing Page

Welcome to the **QuickPOS** landing page project! This is an interactive, visually stunning marketing page built with a modern glassmorphism aesthetic to showcase the ultimate Point of Sale system.

## 🌟 Overview

QuickPOS is designed to be "The Last POS System You'll Ever Need." The landing page is meant to highlight the application's sleek design, fast performance, and robust features like real-time inventory tracking, multi-store management, and advanced analytics.

This repository focuses strictly on the front-end landing page, which features:
- **Hero Section:** An engaging introduction with a simulated, floating UI dashboard built with a glowing glassmorphism effect.
- **Features Section:** Detailed layout of POS capabilities.
- **Pricing:** Transparent pricing tiers highlighting plans from Starter to Enterprise.
- **Contact:** An integrated support and inquiry form.

## 🚀 Features
- **Modern Glassmorphism UI:** Translucent cards, background glows, and floating elements create a deep, layered, and premium user experience.
- **Fully Responsive Design:** The interface adapts perfectly from desktop monitors down to mobile screens.
- **Micro-Animations:** Hover effects, pulse animations, and floating elements add life to the interface.

## 🛠️ Technologies Used
- **HTML5:** Semantic structure.
- **CSS3:** Custom variables, flexbox, grid, animations, and backdrop-filter for glass effects.
- **PHP:** Server-side logic for the contact form and component inclusion (header/footer).
- **Vanilla JavaScript:** Interactive mobile navigation.

## 📦 Setup & Installation

To run this landing page locally, you need a local web server with PHP support (such as MAMP, XAMPP, or WAMP).

### Local Server Configuration (MAMP)
- **Apache Port:** `8888`
- **MySQL Port:** `3308`

1. Clone the repository into your web server's root directory (e.g., `htdocs` for MAMP/XAMPP):
   ```bash
   git clone https://github.com/MAhmadThaheem/quickPOS-landing-page.git
   ```
2. Start your local server (Apache & PHP).
3. Open your browser and navigate to the project directory:
   ```text
   http://localhost:8888/quickPOS-landing-page/index.php
   ```
   *(Note: Adjust the port and folder name depending on your local server configuration).*
